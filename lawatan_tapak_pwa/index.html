Maklumat projek

Isi maklumat asas, pegawai, offset level, GPS lawatan dan catatan lawatan.

Gambar lawatan

Ambil gambar, simpan koordinat, masukkan level dan tambah ayat pemerhatian.

GPS ketepatan tinggi akan mengambil beberapa sampel dan memilih bacaan dengan akurasi terbaik. Jika GPS tidak tersedia, koordinat dan altitude boleh diisi secara manual.

Peta dan koordinat

Tetapan dan install

Pasang aplikasi ini pada iPhone, Android atau Windows, serta import/export data projek.

Install aplikasi

  1. Publish folder ini ke HTTPS supaya app boleh dipasang tanpa perlu WiFi yang sama.
  2. Windows/Android Chrome atau Edge: tekan butang install pada address bar atau butang Install jika muncul.
  3. iPhone Safari: tekan Share, kemudian Add to Home Screen.

Data projek

Data disimpan dalam peranti ini menggunakan IndexedDB. Export JSON boleh digunakan untuk backup atau pindah peranti.

Login code

Login ini ialah local app lock untuk peranti ini. Untuk akaun cloud sebenar, aplikasi perlukan backend/auth server.
